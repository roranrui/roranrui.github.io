顺时针打印矩阵

解题思路:

顺时针打印矩阵的顺序是 “从左向右、从上向下、从右向左、从下向上” 循环。

因此，考虑设定矩阵的 “左、上、右、下” 四个边界，模拟以上矩阵遍历顺序。

实现:

class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        if (matrix.size() == 0) return {};
        int l = 0, t = 0, r = matrix[0].size() - 1, b = matrix.size() - 1;  // 定义边界
        vector<int> res;
        while (true)
        {
            for (int i = l; i <= r; i++)  // 从左往右
                res.push_back(matrix[t][i]);
            if (++t > b) break; // 推进
            for (int i = t; i <= b; i++)  // 从上往下
                res.push_back(matrix[i][r]);
            if (l > --r) break;
            for (int i = r; i >= l; i--)  // 从右向左
                res.push_back(matrix[b][i]);
            if (t > --b) break;
            for (int i = b; i >= t; i--)  // 从下向上
                res.push_back(matrix[i][l]);
            if (++l > r) break;
        }
        return res;
    }
};

栈的压入、弹出序列

解题思路：
如下图所示，给定一个压入序列 pushed 和弹出序列 popped ，则压入 / 弹出操作的顺序（即排列）是 唯一确定 的。

如下图所示，栈的数据操作具有 先入后出 的特性，因此某些弹出序列是无法实现的。

实现:

class Solution {
public:
    bool validateStackSequences(vector<int>& pushed, vector<int>& popped) {
        stack<int> stk;
        int i = 0;
        for (int num : pushed)
        {
            stk.push(num);
            while (!stk.empty() && stk.top() == popped[i])
            {
                stk.pop();
                i++;
            }
        }
        return stk.empty();
    }
};
