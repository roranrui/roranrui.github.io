爬虫学习（一）

一. 什么是爬虫

网络爬虫是一种按照一定的规则，自动地抓取万维网信息的程序或者脚本。

网络爬虫（又称网络蜘蛛，机器人），就是模拟客户端发送网络请求，接收请求响应，一种按照一定的规则，自动地抓取互联网信息的程序。
只要浏览器能够做的事情，原则上，爬虫都能够做到。

简单来说，爬虫就是自动从网络上收集信息的一种程序，复杂点来说，就是一整套关于数据请求、处理、存储的程序。

原理

发送请求 -> 接收响应 -> 解析网页 (-> 提取链接 -> 发送请求) -> 提取资源 -> 保存资源

爬虫分类

  • 全局爬虫
  • 聚焦爬虫
  • 异步爬虫

二. robots.txt协议

详情可以看这

robots协议
robots协议也叫robots.txt（统一小写）是一种存放于网站根目录下的ASCII编码的文本文件，它通常告诉网络搜索引擎的漫游器（又称网络蜘蛛），此网站中的哪些内容是不应被搜索引擎的漫游器获取的，哪些是可以被漫游器获取的。因为一些系统中的URL是大小写敏感的，所以robots.txt的文件名应统一为小写。

robots.txt应放置于网站的根目录下。如果想单独定义搜索引擎的漫游器访问子目录时的行为，那么可以将自定的设置合并到根目录下的robots.txt，或者使用robots元数据（Metadata，又称元数据）。

robots协议并不是一个规范，而只是约定俗成的，所以并不能保证网站的隐私。

三. http和https协议

http协议

服务器和客户端进行数据交互的一种信息

常用请求头信息

User-Agent: 请求载体的身份标识(如edge浏览器, 代码)

Connection: 请求完毕后是断开连接韩式保持连接

常用响应头信息

Content-Type: 服务器响应回客户端的信息

加密方式

  • 对称密钥加密:

  • 非对称密钥加密

  • 证书密钥加密

https协议

是以安全为目标的HTTP通道，简单讲是HTTP的安全版，即HTTP下加入SSL层，HTTPS的安全基础是SSL，因此加密的详细内容就需要SSL。HTTPS协议的主要作用可以分为两种：一种是建立一个信息安全通道，来保证数据传输的安全；另一种就是确认网站的真实性。

四. requests模块

Requests是⽤Python语⾔编写，基于 urllib，采⽤
Apache2 Licensed 开源协议的 HTTP 库。
它⽐ urllib 更加⽅便，可以节约我们⼤量的⼯作，完全满⾜
HTTP 测试需求。
⼀句话——Python实现的简单易⽤的HTTP库

使用步骤:

  • 指定url
  • 发起请求
  • 获取响应数据
  • 持久化存储

简单示例:

import requests as re

# 爬取搜狗首页页面数据
if __name__ == '__main__':
    sogou = 'https://www.sogou.com/'
    response = re.get(url=sogou)
    page_text = response.text

    #storge
    with open('./sougou.html', 'w+', encoding='utf8') as f:
        f.write(page_text)
    print("爬取完成")

案例一: 网页采集器

知识点:

  • url携带的参数处理
  • User-Agent伪装
import requests as re

# UA伪装 User-Agent
# 门户网站的的服务器回检测对应请求的载体身份标识, 因此要伪装为浏览器
if __name__ == '__main__':
    # 进行UA伪装: 将对应的UA封装到一个字典中
    headers = {
        'User-Agent': 'Mozilla/5.0(Windows NT 10.0; Win64; x64) AppleWebKit/537.36(KHTML, likeGecko)Chrome/'
                      '101.0.4951.54 Safari/537.36 Edg/101.0.1210.39'
    }
    url = 'https://www.sogou.com/web'
    # 除了url携带的参数: 封装到字典中
    keyWord = input("请输入你要搜索的内容: ")
    param = {
        'query': keyWord
    }
    # 对指定的url发起的请求对应的url是带参数的,并且请求过程中处理了参数
    response = re.get(url=url, params=param, headers=headers)
    data = response.text
    fileName = keyWord + '.html'
    with open(fileName, 'w+', encoding='utf-8') as f:
        f.write(data)

    print(fileName, "爬取成功!")

案例二: 破解百度翻译

知识点:

  • post请求(携带参数)
  • 抓包工具使用 寻找ajax的sug

ajax的工作原理就是通过XmlHttpRequest对象来向服务器发出异步请求，从服务器中获得数据，然后用Javascript来操作DOM从而更新局部页面。

实现:

import requests as re
import json

if __name__ == '__main__':
    # 进行UA伪装
    headers = {
        'User-Agent': 'Mozilla/5.0(Windows NT 10.0; Win64; x64) AppleWebKit/537.36(KHTML, likeGecko)Chrome/'
                      '101.0.4951.54 Safari/537.36 Edg/101.0.1210.39'
    }
    word = input("请输入你要查询的单词: ")
    post_url = 'https://fanyi.baidu.com/sug'
    data = {
        'kw': word
    }
    response = re.post(url=post_url, data=data, headers=headers)
    # 服务器响应数据
    dic_obj = response.json()
    fp = open('./word.json', 'w', encoding='utf-8')
    json.dump(dic_obj, fp, ensure_ascii=False)
    print("查询完毕!")

案例三: 爬取哔哩哔哩我的追番数据

实现:

import requests as re
import json

if __name__ == '__main__':
    # 进行UA伪装
    headers = {
        'User-Agent': 'Mozilla/5.0(Windows NT 10.0; Win64; x64) AppleWebKit/537.36(KHTML, likeGecko)Chrome/'
                      '101.0.4951.54 Safari/537.36 Edg/101.0.1210.39'
    }
    url = 'https://api.bilibili.com/x/space/bangumi/follow/list'
    params = {
        'type': '1',
        'follow_status': '0',
        'pn': '1',
        'ps': '15',
        'vmid': '30024259',
        'ts': '1652447014110',
    }
    response = re.get(url=url, params=params, headers=headers)
    dic_obj = response.json()
    fp = open('./bangumi.json', 'w', encoding='utf-8')
    json.dump(dic_obj, fp, ensure_ascii=False, indent=4)
    print("爬取成功!")
